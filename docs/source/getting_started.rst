## For most Python users

As of now the project isn't on PyPI, so you'll have to install it from source. 

1. Fork the project on GitHub
2. Clone it to your computer
3. In your terminal, run:

```
python -m venv .venv
source .venv/bin/activate
pip install '.[test]'
```
